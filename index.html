Eclipse Software Sites

This site hosts eclipse features/plugins that are based on the ABAP Development Tools (ADT)

Production Update sites

  • For Eclipse 2019-06 (4.12), use https://eclipse.devepos.com/latest

Development Update sites

Note:
The Development Update sites will most likely host unstable features, so use at your own discretion
  • For Eclipse 2020-09 (4.17), use https://eclipse.devepos.com/dev

Procedure for using the update sites

  1. In Eclipse, choose in the menu bar Help > Install New Software...
  2. Enter the URL of the Software site (see above).
  3. Press Enter to display the available features.
  4. Select the features you want to install.
  5. On the next wizard page, you get an overview of the features to be installed. Choose Next.
  6. Confirm the license agreements and choose Finish to start the installation.
